#### 比赛Tips：

###### Dev调试功能

工具 --> 编译选项 --> 代码生成/优化 --> 连接器 --> 产生调试信息

###### Dev代码生成改为c++11

工具 --> 编译选项 --> 代码生成/优化 --> 代码生成 --> 语言标准

###### Dev语言修改

工具 --> 环境选项 --> 语言

Dev取消括号自动补全

工具 --> 编辑器选项 --> 代码补全 --> 完成符号

###### Dev调试：

- F5调试快捷键，F11直接运行

###### 调试按钮

- 添加查看：监视变量

- 下一步：跳过自定义函数一步一步执行，F7
- 单步进入：进入自定义函数一步一步执行, F8
- 跳过：从当前断点跳到下一个断点执行, S

###### 处理存在空格的字符串：

```C++
	char a[100];
	fgets(a,100,stdin);
	string str = a;
	str.pop_back();	//去掉换行符
```
